# 小简-MD2CV 简历工作台 打包指南

本指南详细说明了如何从源码构建 Windows `.exe` 安装包。

## 1. 环境准备

确保您的 Windows 系统已安装以下环境：
- **Node.js** (推荐 v18+)
- **Rust & Cargo** (最新版本)
- **NSIS** (Tauri 打包器依赖，打包时会自动尝试下载)

## 2. 标准构建流程

在项目根目录下，依次运行以下命令：

```powershell
# 1. 安装依赖
npm install

# 2. 前端预检查与打包
npm run build

# 3. 使用 Tauri 构建安装包
npm run tauri build
```

构建完成后，安装包将生成在：
`src-tauri/target/release/bundle/nsis/小简-MD2CV简历工作台_0.1.0_x64-setup.exe`

## 3. 常见问题：WiX 打包错误 (Light.exe)

### 现象
报错信息包含：`failed to run ... light.exe` 或 `failed to bundle project`。

### 原因
本项目的 `productName` 包含中文（“小简-MD2CV简历工作台”）。Tauri 默认使用的 WiX 工具集（生成 `.msi`）不支持在输出路径或文件名中出现非 ASCII 字符。

### 解决方法
我们已将打包目标锁定为 **NSIS**。NSIS 跨平台支持更好，且完美支持 Unicode，可以保留中文产品名称并生成 `.exe` 安装包。

相关配置位于 `src-tauri/tauri.conf.json`：
- `bundle.targets`: 设为 `["nsis"]`
- `bundle.windows.nsis`: 禁用了语言选择器并设置为机器级安装

## 4. 其他注意事项
- **版本一致性**：若看到 `mismatched versions` 警告，请在 `src-tauri` 目录下运行 `cargo update`。
- **图标更新**：若需修改安装包图标，请替换 `src-tauri/icons` 目录下的相关文件。
